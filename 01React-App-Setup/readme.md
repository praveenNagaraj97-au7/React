# React App Setup

```
Babel Packages
    npm i -D @babel/node @babel/core @babel/preset-env @babel/preset-react

Webpack Packages
    npm i -D webpack webpack-cli webpack-dev-server

Loaders
    npm i -D babel-loader css-loader style-loader sass-loader file-loader

plugins
    npm i -D html-webpack-plugin @babel/plugin-proposal-class-properties

react
    npm i react react-dom
```
